# 异常合同：软件开发服务采购合同

甲方：甲方科技有限公司
乙方：乙方软件有限公司

一、项目内容
乙方为甲方开发合同在线编辑与审查系统。具体功能范围以双方后续沟通记录、会议纪要或即时通讯记录为准，新增需求费用由双方酌情安排。

二、合同价款与付款
合同金额暂定为人民币 100,000 元。付款时间、付款比例、发票要求和付款条件由双方后续协商确定。

三、交付与验收
乙方交付日期可根据实际情况顺延。验收标准由双方后续沟通确定，甲方应尽快安排验收。

四、违约责任
任一方违约的，双方友好协商解决。因政策变化、人员变动、供应商安排等原因导致延迟或不能履行的，相关方不承担违约责任。

五、合同期限
本合同自双方确认后生效，项目完成后终止。续期、解除通知期限及资料移交安排由双方另行协商。

六、争议解决
发生争议后，双方另行确定处理方式。
